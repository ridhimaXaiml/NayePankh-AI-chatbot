# NayePankh-AI-chatbot
AI-powered NGO chatbot for NayePankh Foundation built using Python. It provides information about NGO activities, volunteering, donations, and support through a simple rule-based conversational system.

# NayePankh Foundation AI Chatbot 🤖
## Features
- Interactive chat interface
- Personalized greeting based on time
- Answers questions about:
  - NGO information
  - Volunteering
  - Donations
  - Internships
  - Events and support programs
- Handles multiple user queries
- Simple rule-based NLP approach

## Technologies Used
- Python
- Dictionary-based NLP logic
- Datetime module

## Purpose
The aim of this project is to make NGO information easily accessible and improve communication between the organization and users.
